Основные команды

.sub

Подписаться на каналы.

Примеры ссылок:

  • @username
  • t.me/channel
  • t.me/+invite_code

.ref

Отправить реферальные запросы с поддержкой капчи.

Пример:

.ref https://t.me/BestRandom_bot?start=lot_join_nc3284492p2e9f36d9b672aa75d893df05497bf0ef

Ожидание до 120 секунд, не более 3 попыток.

.refk

Обработка реферальных ссылок из поста с реферальным запросом.

Пример: .refk https://t.me/emapubg/1914

.run

Выполнить действия из сообщения с логированием.

Пример: .run https://t.me/n3koglai/645

.unsub

Отписаться от каналов.

.unsuball

Полная очистка подписок (кроме белого списка).

Примеры:

  • .unsuball 100
  • .unsuball all

.aw

Добавить каналы/чаты в белый список.

Пример: .aw 123456789

.whitelist

Вывести текущий белый список с информацией о каналах.

.giveshare

Включить/выключить автоматическое участие в розыгрышах.

.pupdate

Проверить обновление модуля и обновить, если необходимо.

Группы с настройками задержки

.autogroup

Два варианта использования:

Формат 1: .autogroup <группа> <множитель> <chat_id>

Пример: .autogroup 1 5.0 2328066448

Формат 2: .autogroup <задержка> <chat_id>

Пример: .autogroup 11.39 2328066448

.setg

Установить один из пресетов задержки между действиями.

  • 1: 5 секунд
  • 2: 30 секунд
  • 3: 45 секунд
  • 4: 60 секунд
  • 5: 75 секунд

Пример: .setg 1

Информация об аккаунте

.getcode

Запросить код верификации.

.getinfo

Получить информацию об аккаунте: номер, ID, дата регистрации, лимиты подписок.

.getnumber

Запросить номер аккаунта.

Кастомизация аккаунта (мут, сник, тайм)

.time

Скрыть время входа в аккаунт (настройки конфиденциальности).

.mutecmd

Управление уведомлениями.

0 – включить мут (отключить уведомления)

1 – отключить мут (включить уведомления)

.snick

Копировать профиль случайного пользователя из указанного чата или текущего.

Примеры: .snick @chatusername или .snick t.me/yourchat

⚙️ Настройки .cfg

Путь: Установленные → Minamotosoft2

Параметр Описание
api_key Ключ 2captcha
delay Базовая задержка (секунды)
log_chat_username Чат для логов (@username)
watcher_on Автопоиск розыгрышей

log_chat_id ID чата для ошибок
success_log_chat_id ID чата для успешных операций

Изменяйте параметры через: .config MinamotoSoftV2

Возможные ошибки и их решения

Ошибка: no query results

Эта ошибка указывает на то, что при вызове команды нет результатов для указанного unit_id, что может быть вызвано сломанным inline ботом.

Решение: необходимо переустановить inline бота, используя команду:

.ch_hikka_bot new bot

Пример: .ch_hikka_bot sdkadksadkds_bot / MAXIMKAHIKKA_BOT
